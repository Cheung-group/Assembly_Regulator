# INO80_Assembly
written by Jiayi Wang 2025.5

This directory contain core codes for the INO80 assembly projects, indlucing
Int_convergence: inversely determine interaction energies from subunits' contact frequencies inferred from the cross linking experment.

Mu_convergence: codes for individual convergence, which inversely determine chemical potentials from subunit's abundance at a given volume fraction (set in newMu.m). 

N_mu_diagram: codes to map out the chemical potential - abundance curves for each subunit.

N_perturbation: codes for analyzing change in accessible volume and potential energy upon particle number perturbation in canonical ensemble.
