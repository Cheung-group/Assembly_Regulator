# Graph Statistics

This folder contains Python code for analyzing the network structure of graphs
generated by `graphing/`.

## Purpose

The scripts calculate frame-averaged coordination statistics from edgelist
representations of INO80 simulation trajectories. These statistics complement
the thermodynamic particle-number perturbation analysis by quantifying how
subunit abundance changes affect overall clustering.

## Key Files

- `basestatistics.py`: computes baseline mean coordination number and
  coefficient of variation for the empirical-abundance trajectory.
- `perturbedstatistics.py`: computes the same quantities for perturbed
  trajectories and reports differences relative to the baseline.

## Metrics

For each sampled frame:

- Degree centrality is converted back to particle coordination number.
- Isolated particles are added back as zero-coordination entries because they
  are omitted from NetworkX edgelist files.
- The script records the mean coordination number and the coefficient of
  variation across all particles.

The final reported values are averages over sampled frames.

## Workflow

1. Generate baseline and perturbed edgelists using `graphing/`.
2. Run `basestatistics.py` to compute `basemean` and `basecoeff`. `basemean` is the frame-averaged mean coordination number for the baseline simulation where all INO80 subunits are at empirical abundance; `basecoeff` is the frame-averaged coefficient of variation of coordination number for that same baseline simulation.
4. Enter those baseline values in `perturbedstatistics.py` if they differ from
   the values already included.
5. Run `perturbedstatistics.py` for the desired subunits and abundance
   perturbations.

## Outputs

`perturbedstatistics.py` appends results to files named by subunit, for example
`subunit_<type>.txt`. Each entry records the difference in mean coordination
number and the difference in coefficient of variation relative to the baseline.

## Notes

- The included paths use `/home/[PATH_TO_FILE]/` placeholders and must be
  updated before running.
- `perturbedstatistics.py` contains hard-coded empirical and perturbed abundance
  values in `abd_matrix`.
- The baseline values currently embedded in `perturbedstatistics.py` are:
  `basemean = 0.9591778810863715` and
  `basecoeff = 1.8069102818895462`.
